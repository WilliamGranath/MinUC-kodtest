# UC Demo

## Hur man kör projektet

1. Klona repositoryt:

   ```sh
   git clone <repo-url>
   cd uc-demo
   ```

2. Installera beroenden:

   ```sh
   npm install
   ```

3. Starta utvecklingsservern:

   ```sh
   npm run dev
   ```

4. Öppna webbläsaren och besök `http://localhost:5173/` för att se sidan.
